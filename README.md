# my_rpg
Epitech Project 'my_rpg', a rogue-like dungeon crawler developed in csfml (not coding style compliant!)

Controls:
'wasd': move
arrow keys: shoot
E/M1: interact

Room legend (for room creation):
0-8: obstacles
'/\-_[]()': walls
a-f: regular enemies
A-D: stronger version of enemies a-d
!: top door
?: bottom door
~: left door
=: right door
